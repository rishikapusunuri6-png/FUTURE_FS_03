# 💄 RS Beauty Salon Website

A modern, responsive beauty salon website built using HTML, CSS, JavaScript, and Bootstrap.  
This project is created as part of a Full Stack Web Development internship task to simulate a real-world business website.

---

## 🌟 Project Overview

RS Beauty Salon website is designed for a real local salon business to improve its online presence, attract more customers, and provide easy access to services, pricing, and contact details.

It demonstrates how developers build websites for real clients in freelancing and agency work.

---

## 🎯 Objective

- Create a professional website for a local beauty salon
- Improve online visibility of the business
- Showcase services and pricing clearly
- Make customer communication easier
- Simulate real freelance project experience

---

## ✨ Features

- 💆 Facial, Hair Styling, Makeup services
- 💍 Bridal package section
- 📸 Modern image gallery with hover effects
- 👩 About Us section
- 📞 Contact form with validation
- 📱 WhatsApp direct chat button
- 🎨 Clean and modern UI design
- ⚡ Fully responsive (mobile + desktop)
- 🧭 Smooth navigation using Bootstrap

---

## 🛠️ Technologies Used

- HTML5  
- CSS3  
- JavaScript (Vanilla JS)  
- Bootstrap 5.3.3  
- Font Awesome Icons  

---

## 🚀 How to Run the Project

1. Download or clone this repository

4. No installation required
